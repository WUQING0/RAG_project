# RAG 项目示例知识库

这个项目提供一个对话、图片输入和 RAG 知识库检索的完整演示。

主要能力：

- 上传 txt、md、csv、json、pdf 文档并自动切分为知识片段。
- 为片段生成向量并按照用户问题检索最相关内容。
- 聊天接口会把检索片段作为上下文发送给大模型。
- 配置视觉模型后，聊天接口可以同时携带图片和文本问题。
- 没有配置 API Key 时，系统会进入本地演示模式，仍然可以测试上传、检索和引用展示。

建议使用流程：

1. 先在右侧知识库面板上传业务文档。
2. 在左侧对话框输入问题，并保持 RAG 开关开启。
3. 如果需要图片理解，添加图片后提问，并配置支持视觉能力的模型。

环境变量：

- APP_AI_API_KEY：模型服务 API Key。
- APP_AI_BASE_URL：OpenAI 兼容接口地址。
- APP_AI_CHAT_MODEL：普通聊天模型。
- APP_AI_VISION_MODEL：多模态视觉模型。
- APP_AI_EMBEDDING_MODEL：向量模型。
